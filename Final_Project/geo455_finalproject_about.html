About This Webmap
Webmap Overview

This interactive webmap explores the history of tornadoes rated EF3 - EF5 that have touched down across the United States. It has been designed as an educational tool for late elementary and middle school students, helping young learners understand the power, frequency, and distribution of large scale tornadoes across the U.S.

The map is made up of three thematic layers: Tornado Intensity (colored by EF scale rating), Tornado Frequency (a heatmap showing where tornadoes occur most often), and Tornado Impact (proportional symbols showing fatalities and injuries). A time slider allows users to filter tornadoes by year.

How to Use the Map
  • Use the layer control on the map to switch basemaps and toggle tornado layers.
  • Click on any tornado marker to open a popup and see its details in the side panel.
  • Drag the year slider in the side panel to filter tornadoes by a specific year.
  • Click the globe icon on the map to return to the default U.S. view.
Data Sources
NOAA Storm Events Database - ncei.noaa.gov/stormevents
NOAA was the primary dataset for this Webmapping project. The tornado event records were downloaded as CSV files and filtered to include only EF3–EF5 tornadoes (2010-2020), due to data constraints. The primary fields used include: EF rating, start and end coordinates, date, fatalities, injuries, path length, and property damage.
NOAA U.S. Tornadoes (NCEI) - ncei.noaa.gov/access/monitoring/tornadoes
The annual tornado count and fatality statistics from this source was used as reference material for this webmap.
Tornado Archive - tornadoarchive.com
A detailed historical database of U.S. tornadoes, which was used as an additional reference to verify event records prior to the creation of the webmap.
Data Processing

The initial CSV data from the NOAA database was both imported and processed in ArcGIS Pro. The XY Table to Line tool was used to convert start and end coordinates into a feature class showing each tornado's path. This was then projected to Albers Equal Area Conic for length calculations, and reprojected back to WGS 1984 to be exported as a GeoJSON file.
