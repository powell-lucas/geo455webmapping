Lab 2: My first interactive map

About this map

This is the final result of the Leaflet Quick Start Guide. The map shows the location of my favorite overlook in the La Crosse Bluffs. The map demonstrates Leaflet basics, such as setting up a Leaflet map and adding markers and popups. The placement and styling of the title and description boxes is done using CSS. Check out the source code of this page to see how.
